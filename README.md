# CRM システム

Php laravel ベースの包括的な顧客関係管理 (CRM) システムです。

## 概要

このプロジェクトは、企業の顧客管理、製品管理、店舗管理、店員管理を統合的に行うためのWebアプリケーションです。Bootstrap 5を使用したモダンで使いやすいUIを提供します。

## 主要機能

### ✅ 認証機能
- ユーザーログイン/ログアウト
- セッション管理
- 認証必須ページ保護

### ✅ 顧客管理
- 顧客登録・編集・削除
- 顧客一覧表示（検索・ページネーション対応）
- 顧客詳細情報表示

### ✅ 製品管理
- 製品登録・編集・削除
- カテゴリ別製品管理
- 在庫状況管理
- 価格表示機能

### ✅ 店舗管理
- 店舗登録・編集・削除
- 営業状況管理
- 店舗詳細情報表示
- 営業年数自動計算

### ✅ 店員管理
- 店員登録・編集・削除
- 役職・部署管理
- 勤続年数自動計算
- 給与情報管理

## 技術仕様

- **フレームワーク**: laravel 8.0
- **言語**: php7.3.11
- **データベース**: MySQL
